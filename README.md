# react-webpack-simple-startkit
A simplest app built by [Webpack2](http://webpack.js.org/) using [React](https://facebook.github.io/react/)

## Installation
* Make sure you have a global support for webpack
* Run `npm install`
* Run `npm run dev`
* Open `localhost:8080`
* Enjoy the React HMR
