# react-webpack-simple-startkit
A simple app build by [Webpack](http://webpack.github.io/) using [React](https://facebook.github.io/react/)

## Installation
* Make sure you have a global support for webpack
* Run `npm install`
* Run `npm run dev`
* Open `localhost:8080`
